## PART 2: VISUAL DATA - PERFORMANCE OPERATIONAL DASHBOARD (POD) (3:00 - 12:00)

### 2.1 Understanding the POD (3:00 - 3:30)

**[SCREENSHOT: Full POD view]**

"The Performance Operational Dashboard, or POD, is the heart of the FIS Dashboard. This is where you'll see comprehensive visual analytics of your workroom performance data.

The POD displays:
- **Heatmap visualization** - Color-coded workroom performance
- **Performance metrics** - Key indicators at a glance
- **Geographic map** - Workroom locations and performance
- **Data tables** - Detailed breakdowns and rankings"

### 2.2 The Heatmap - Core Visualization (3:30 - 5:00)

**[SCREENSHOT: Heatmap with color-coded workrooms]**

"The heatmap is the primary visualization tool. Each workroom is displayed as a card with color-coded performance indicators.

**[ZOOM IN: Single workroom card]**

Each workroom card shows:
- **Workroom name**
- **WPI Score** - Workroom Performance Index (weighted score)
- **Performance label** - Top Performing, Moderate, or Critical
- **Key metrics** - LTR, Labor PO $, Vendor Debits, Store Mix
- **Risk indicators** - Operational and financial risks

**[SCREENSHOT: Color coding legend]**

The color coding works like this:
- **Green** - Top Performing (WPI ≥ 85)
- **Yellow** - Moderate (WPI 70-84)
- **Red** - Critical (WPI < 70)

This visual system makes it easy to quickly identify which workrooms need attention."

### 2.3 WPI Score Calculation (5:00 - 5:45)

**[SCREENSHOT: WPI score breakdown]**

"The Workroom Performance Index, or WPI, is a weighted score calculated from multiple factors:

- **50% LTR Performance** - Labor-to-Revenue ratio
- **5% Details Cycle Time** - Time to complete detail work
- **13% Job Cycle Time** - Time to complete jobs
- **14% Work Order Cycle Time** - Total work order processing time
- **8% Reschedule Rate** - Frequency of rescheduled work
- **10% Vendor Debits** - Vendor debit exposure

This comprehensive scoring system gives you a complete picture of workroom performance."

### 2.4 Top Performing Workrooms Table (5:45 - 6:30)

**[SCREENSHOT: Top Performing Workrooms table]**

"Below the heatmap, you'll find the 'Top Performing Workrooms' table. This shows the top 15 workrooms ranked by their WPI score.

- **LTR Performance** - Labor-to-Revenue percentage
- **Labor PO $** - Total labor purchase order volume

This table helps you identify your best performers and learn from their success."

## PART 3: SURVEY DATA (9:30 - 15:00)

### 3.1 Understanding Survey Data (9:30 - 10:00)

**[SCREENSHOT: Survey Misc page]**

"Survey data is a crucial component of the FIS Dashboard. It provides customer feedback and installer performance metrics.

To access survey data, click on 'Survey Misc' in the sidebar navigation.

**[SCREENSHOT: Survey data overview]**

- **Craft Scores** - Craftsmanship ratings

- **Installer Information** - Installer names and performance
- **Company Data** - Company information from surveys"

### 3.2 Survey Data Upload (10:00 - 10:45)

**[SCREENSHOT: File upload section]**

"Survey data is uploaded separately from visual data. This allows you to update survey information independently.

**[SCREENSHOT: Survey file upload button]**

As an admin, you'll see two upload sections:
1. **Visual Data Upload** - For T1/T2 scorecard operational data
2. **Survey Data Upload** - For survey and customer feedback data

**[CLICK: Survey Data Upload]**

Click 'Choose Survey Data File' and select your survey Excel file. The system will:
- Parse the survey data
- Extract LTR, Craft, and Professional scores
- Extract installer names and company information
- Store the data separately from visual data
- Display a progress indicator during upload"

### 3.3 Survey Data Integration with Heatmap (10:45 - 11:30)

**[SCREENSHOT: Heatmap with survey data]**

"One of the powerful features of the dashboard is how survey data integrates with the heatmap.

**[SCREENSHOT: Workroom card with survey LTR]**

When survey data is available, the heatmap uses:
- **Survey LTR scores** instead of calculated LTR percentages
- This provides more accurate performance ratings
- Survey data takes priority over calculated values

**[SCREENSHOT: LTR Performance rating]**

You'll see survey-based LTR scores displayed in the workroom cards with labels like:
- **Excellent** - LTR > 9.0
- **Good** - LTR 8.0-9.0
- **Moderate** - LTR 7.0-8.0
- **Poor** - LTR 6.0-7.0
- **Critical** - LTR < 6.0"

## TROUBLESHOOTING

### Common Issues:

**Can't sign in?**
- Ensure you're using a Microsoft account
- Check if your account has been authorized
- Request access if you see the access required message

**No data showing?**
- Verify data has been uploaded (admin function)
- Check your workroom filter settings
- Ensure you're viewing the correct page

**Survey data not appearing?**
- Confirm survey data file was uploaded successfully
- Check that the file format matches requirements
- Verify data is associated with the correct workrooms
