# FIS POD Chat - Quick Guide

## Overview
The chat widget allows you to respond to conversations started by Lowe's Pro Connect. The chat appears as a floating button in the bottom-right corner of all pages.

---

## Accessing Chat

### Opening Chat
1. Look for the **green chat button** in the bottom-right corner
2. Click the button to open the chat widget
3. The chat window will appear

### Closing/Minimizing
- Click the **X** button to close
- Click the **minimize** button (dash) to minimize
- Click the chat button again to reopen

---

## Using Chat

### Step 1: Select a Conversation
- When you open chat, you'll see a list of conversations
- Conversations are started by Lowe's Pro Connect
- Click on any conversation to open it

### Step 2: View Conversation Details
- Click **"Show details"** to see:
  - Quote/IMS number
  - Flooring category
  - What they're questioning
  - User information

### Step 3: Send Messages
1. Type your message in the input field at the bottom
2. Click **Send** button (or press Enter)
3. Your message appears on the right (green)
4. Messages from Lowe's team appear on the left (white)

---

## Chat Features

### Online Status Indicator
- **Green dot** - Shows when a user is online
- Located next to the chat title in the header
- Green pulsing dot = user is currently active
- Helps you know if someone is available to chat

### Message Display
- **Your messages** - Green bubbles on the right
- **Lowe's messages** - White bubbles on the left
- **Timestamps** - Shows when each message was sent
- **Profile photos** - Shows user photos when available

### Auto-Refresh
- New messages appear automatically
- Chat updates every 10 seconds
- No need to refresh manually

### Conversation List
- Shows all active conversations
- Sorted by most recent activity
- Click any conversation to switch

✅ **Include details** - Reference quote numbers when needed  
✅ **Check details** - Review conversation details before responding  

---

## Troubleshooting

### Chat Not Appearing?
- Make sure you're logged in
- Check if chat button is visible (bottom-right corner)
- Refresh the page

### Can't Send Messages?
- Ensure you've selected a conversation
- Check your internet connection
- Try refreshing the page

### Messages Not Showing?
- Wait a few seconds (auto-refresh every 10 seconds)
- Check if conversation is selected
- Refresh the page if needed
